unstable_getAgentDefaultAccessAll#

▸ unstable_getAgentDefaultAccessAll(aclDataset: unstable_AclDataset): unstable_AgentAccess

Defined in acl/agent.ts:251

Given an ACL LitDataset, find out which access modes it provides to specific Agents for the associated Container Resource's child Resources.

Keep in mind that this function will not tell you:

  • what access arbitrary Agents might have been given through other ACL rules, e.g. public or group-specific permissions.
  • what access arbitrary Agents have to the Container Resource itself (see unstable_getAgentResourceAccessAll for that).

Also, please note that this function is still experimental: its API can change in non-major releases.

Parameters:

NameTypeDescription
aclDatasetunstable_AclDatasetThe LitDataset that contains Access-Control List rules.

Returns: unstable_AgentAccess

Which Access Modes have been granted to which Agents specifically for the children of the Container associated with the given ACL LitDataset.

unstable_setAgentDefaultAccess#

▸ unstable_setAgentDefaultAccess(aclDataset: unstable_AclDataset, agent: WebId, access: unstable_Access): unstable_AclDataset & WithChangeLog

Defined in acl/agent.ts:280

Given an ACL LitDataset, modify the ACL Rules to set specific default Access Modes for a given Agent.

If the given ACL LitDataset already includes ACL Rules that grant a certain set of default Access Modes to the given Agent, those will be overridden by the given Access Modes.

Keep in mind that this function will not modify:

  • access arbitrary Agents might have been given through other ACL rules, e.g. public or group-specific permissions.
  • what access arbitrary Agents have to the Container itself.

Also, please note that this function is still experimental: its API can change in non-major releases.

Parameters:

NameTypeDescription
aclDatasetunstable_AclDatasetThe LitDataset that contains Access-Control List rules.
agentWebIdThe Agent to grant specific Access Modes.
accessunstable_AccessThe Access Modes to grant to the Agent.

Returns: unstable_AclDataset & WithChangeLog
