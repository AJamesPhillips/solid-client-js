Installation

To install the latest stable version of solid-client, run:

npm install @inrupt/solid-client

solid-client will work in Node using CommonJS modules, and in the browser with a bundler like Webpack, Rollup or Parcel.

To access private data#

By default, solid-client will only enables to access public data on Solid Pods. To allow for access to restricted data, you will need to include a library that supports user authentication. If you have solid-auth-client installed and have used it to authenticate the user, solid-client will automatically pick up the authenticated session and include the user's credentials with each request.
